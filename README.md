# Wolf Cleaning — Product Mockups

Interactieve UI mockups voor een SaaS CRM gericht op zelfstandige ruitenwassers (BE/NL).

## Schermen

| Bestand | Omschrijving |
|---------|-------------|
| `index.html` | Overzichtspagina met navigatie naar alle schermen |
| `wolfcleaning_dashboard.html` | Desktop dashboard met KPI's, zoneplanning en klanttabel |
| `wolfcleaning_crm_mockup_v2.html` | Mobile CRM — dagplanning, klantfiche en leads pipeline |
| `wolfcleaning_routeplanner.html` | Interactieve routeplanner met kaart en optimalisatie |

## Live bekijken

Na activatie van GitHub Pages is de site beschikbaar op:  
`https://<jouw-username>.github.io/wolfcleaning-mockups/`

## Tech

Puur HTML/CSS/JS — geen build step, geen dependencies. Alle bestanden werken standalone en zijn responsive (desktop + mobile).

## Context

- **Product**: CRM voor buitendienstwerkers met terugkerende klanten
- **Fase 1**: Ruitenwassers in België en Nederland
- **Stack (target)**: Laravel + React, Google Maps API, PWA
- **Door**: [Conversal BV](https://conversal.be), Affligem
